# 포즈 분석 엔진 (srd_pose_emergency_core) 판단 기준 및 파라미터 정리

현재 최신 코드 기준으로 적용된 가시성, 자세, 움직임, 위험 단계 판단 로직과 조정된 파라미터들을 표로 정리한 문서입니다.

## 1. 가시성 (Visibility) 상태 분류

사람의 몸이 화면에 얼마나 노출되었는지 판단하며, 이는 관절 포인트(Keypoint)의 개수에 따라 결정됩니다.

| 상태명 | 적용 조건 (파라미터 변수명) | 세부 설명 |
| :--- | :--- | :--- |
| `LOW_CONF` | 유효 관절 수 < `low_conf_min_kps` (3) | 신뢰할 수 있는 관절이 3개 미만이라 오탐지로 간주하거나 분석 불가한 상태 |
| `FULL_BODY` | 상체 관절 >= `upper_body_min_kps` (3)<br>+ 골반 관절 감지됨<br>+ 추가 하체 관절 >= `full_body_extra_lower_kps` (1) | 상하체가 충분히 확보된 전신 노출 상태로, 가장 높은 신뢰도의 분석 가능 |
| `UPPER_BODY` | 상체 관절 >= `upper_body_min_kps` (3) | 하체는 가려졌지만 상체(어깨, 팔 등)는 선명하게 잡히는 상태 |
| `PARTIAL` | 위 조건 미달, 그러나 `LOW_CONF`는 아님 | 잔해물에 깔려 신체 일부만 보이는 등 판단이 애매한 부분 노출 상태 |

---

## 2. 자세 (Posture) 상태 분류

카메라에 잡힌 관절들의 각도와 신체 비율을 계산해 사람이 어떤 자세인지 수학적으로 판단합니다. (`FULL_BODY` 기준)

| 상태명 | 임계 파라미터 (변수명 및 기준값) | 적용 조건 및 상황 |
| :--- | :--- | :--- |
| `LYING`<br>(누움) | `lying_aspect_ratio` (>= 2.00)<br>`lying_torso_angle_deg` (>= 72.0도) | 몸의 바운딩 박스 가로/세로 비율이 2.0배 이상 길쭉하거나,<br>척추 각도가 바닥선 기준 72도 이상 크게 꺾여 누운 상태 |
| `COLLAPSED`<br>(붕괴/쓰러짐) | `collapsed_torso_angle_deg` (>= 60.0도)<br>`collapsed_shoulder_tilt_deg` (>= 45.0도)<br>`collapsed_head_drop_ratio` (>= 0.78) | *LYING까진 아니여도 아래 중 하나에 해당:*<br>1) 허리가 60도 이상 크게 꺾임<br>2) 어깨가 45도 이상 한쪽으로 쏠림<br>3) 고개가 어깨에 78% 이상 기절하듯 파묻힘 |
| `LEANING`<br>(기울어짐) | `leaning_shoulder_tilt_deg` (>= 25.0도)<br>`leaning_head_drop_ratio` (>= 0.55) | *완전한 쓰러짐은 아니지만 불안정한 상태:*<br>1) 어깨선이 25도 이상 비틀리거나 처짐<br>2) 고개가 어깨 방향으로 55% 이상 내려옴 |
| `NORMAL`<br>(정상) | 상기 위험 조건들에 아무것도 해당되지 않음 | 꼿꼿하게 서 있거나 바르게 앉은 평범하고 안정적인 자세 |

> **참고:** `UPPER_BODY` 일때는 골반(척추) 각도를 구할 수 없으므로 `torso_angle`은 판정 기준에서 제외됩니다.

## 3. 움직임 (Motion) 크기 분류

이전 프레임과의 픽셀 이동량을 사람의 키 기준 비율로 추산하여, 움직임의 활발함을 판별합니다. (최근 수정본 반영)

| 상태명 | 임계 파라미터 (변수명 및 기준값) | 적용 조건 및 상황 |
| :--- | :--- | :--- |
| `ACTIVE`<br>(활발함) | `motion_active_smooth` (>= 0.020)<br>`motion_active_upper` (>= 0.025) | N프레임 평균 움직임 폭이 충분히 크거나(0.020),<br>순간적인 상체 허우적거림이 매우 클 때(0.025) |
| `LOCAL_ONLY`<br>(일부분 요동) | `motion_local_only_upper` (>= 0.015)<br>`motion_local_only_core` (<= 0.010) | 시스템 코어(가슴/골반)는 고정되어 안 움직이는데(0.010 이하),<br>팔뚝이나 무릎 등 국소 부위만 크게 움직임(0.015 이상). (주로 깔림 의심) |
| `LOW`<br>(미세 움직임) | `motion_low` (>= 0.008) | `ACTIVE` 하진 않지만, 숨쉬기나 가벼운 뒤척임, 몸 떨림수준의 수치가 확인될 때 |
| `NONE`<br>(꼼짝 않음) | 이동량 < `motion_low` (0.008) | 카메라 노이즈 보정을 뺀 실제 유의미한 움직임이 전혀 감지되지 않는 상태 (기절/사망 의심) |

## 4. 최종 위급 단계 (Emergency Level)와 지속 시간 제어

사람이 잠시 허리를 숙이거나 멈춰 서는 일상적 행동을 위험으로 오인하지 않도록, **"해당 위험 상태가 얼마나 누적/지속되었는가(Time Delay)"**를 두고 방어적으로 단계를 올립니다.

### 시간 누적 파라미터(초)
- `analyzing_sec` = **1.5초** : 카메라에 처음 발견된 직후 섣부르게 분석하지 않는 대기 및 데이터 버퍼 수집 기간
- `caution_sec` = **4.5초** : 주의 발령(`CAUTION`)을 울리기 위한 최초 최소의 비정상 자세/멈춤 유지 시간
- `warning_sec` = **5.5초** : 1차 경고(`WARNING`) 조건 달성 시간 (기절 초기로 간주)
- `critical_sec` = **7.0초** : 심각한 초위급(`CRITICAL`) 상황을 확정 짓는 데드라인

### 위급 등급 결정표 (FULL_BODY 기준 대표적 시나리오)

| 상태 조합 시나리오 (Posture + Motion) | 해당 상태 누적(지속) 필요 시간 | 발효되는 위급 단계 |
| :--- | :--- | :--- |
| `LYING` 또는 `COLLAPSED` 이면서 미동도 없음(`NONE`) | **7.0초** (`critical_sec`) 이상 | 🚨 **CRITICAL** (즉각 구조조치 필요) |
| `LYING` 또는 `COLLAPSED` 인데 살짝 꼼지락거림(`LOW`, `NONE`) | **5.5초** (`warning_sec`) 이상 | ⚠️ **WARNING** |
| `LEANING` 상태로 멍하니 기댐(`LOW` 또는 `NONE`) | **4.5초** (`caution_sec`) 이상 | 🟡 **CAUTION** |
| 자세는 `NORMAL`이지만 선 채로 조각상처럼 굳어있음(`NONE`) | **5.5초** (`warning_sec`) 이상 | 🟡 **CAUTION** |
| `LOW` 수준의 낮은 활동성만 계속 감지됨 | **4.5초** (`caution_sec`) 이상 | 🟡 **CAUTION** |
| 그 외 짧은 시간 내 자세 복귀 또는 `ACTIVE` 한 움직임 | - | 🟢 **NORMAL** |
